# Настройка загрузки изображений для вопросов

## После деплоя на сервер выполните:

```bash
cd ~/public_html/bot

# Создать директорию для изображений
mkdir -p storage/app/public/questions

# Создать симлинк для публичного доступа (если еще не создан)
php artisan storage:link

# Установить права доступа
chmod -R 775 storage/app/public/questions
```

## Проверка:

После настройки изображения будут доступны по адресу:
`https://ваш-домен.ru/storage/questions/имя_файла.jpg`

## Использование:

1. При создании/редактировании вопроса выберите тип "С изображением"
2. Загрузите файл изображения (JPEG, PNG, JPG, GIF, WEBP до 10MB)
3. Или укажите URL изображения
4. Или укажите File ID из Telegram

Файлы автоматически сохраняются в `storage/app/public/questions/`
